# Url_shortener in Golang

## Steps to run

- Run 
```
sql.connection.go
``` 
This includes connection to database and tables.

- Then run
```
shortner.go
```
This file includes functionalities to get 
the short url.

- Finally, run
```
Data/data.go
data.go
```
Here, we can store our Long url and short url
and retrieve them accordingly.
